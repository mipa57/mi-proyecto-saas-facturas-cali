=== CLAUDE CODE - LECCIÓN 5 - FLUJO DE TRABAJO ===

EXPLORE ? PLAN ? CODE ? COMMIT

EXPLORE: Claude lee sin modificar nada
PLAN: Plan detallado antes de escribir código
      ? MEJOR MOMENTO PARA CORREGIR ?
CODE: Ejecuta con aprobación en cada paso
      Tips:
      - Define criterio de éxito
      - Agrega herramientas
      - Incluye tests
      - Errores repetidos ? guardar en CLAUDE.md
COMMIT: 
      - Prueba tú mismo
      - Subagente revisa (ojos frescos)
      - Claude genera mensaje de commit
      - Sube a GitHub

REGLA DE ORO:
Nunca pidas código sin explorar y planificar primero

1. "Explora este código que te pego"
2. "Planifica cómo agregar X función"
3. "Ahora escribe el código paso a paso"
4. "Revisa el resultado final"

=== CLAUDE CODE - LECCIÓN 11 - HOOKS ===

QUÉ SON:
Reglas deterministas que SIEMPRE se ejecutan
Sin excepciones, sin importar el contexto

EVENTOS:
PreToolUse ? antes de actuar (puede BLOQUEAR)
PostToolUse ? después de actuar
UserPromptSubmit ? al enviar prompt
Stop ? cuando Claude termina
Notification ? cuando Claude notifica

CÓDIGOS PreToolUse:
0 ? procede
2 ? BLOQUEADO (Claude sabe por qué)
otro ? error visible, no bloquea

PARA MI SAAS:
? PostToolUse: formatear Python automático
? PostToolUse: validar NIT colombiano
? PreToolUse: bloquear rm -rf
? Stop: notificar por WhatsApp/email

CONFIGURACIÓN:
