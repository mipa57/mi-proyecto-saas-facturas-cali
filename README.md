# 🧾 SaaS de Facturas - Cali, Colombia

Proyecto de automatización de procesamiento 
de facturas electrónicas para PyMEs en Cali.

## 👨‍💻 Desarrollador
Miguel Gustavo Bejarano Patiño
Cali, Valle del Cauca, Colombia

## 🎯 Objetivo
Automatizar la extracción de datos de facturas 
PDF usando Python y Claude API, dirigido a 
PyMEs colombianas.

## 🛠️ Stack Tecnológico
- Python 3.10
- pdfplumber (lectura de PDFs)
- Claude API (extracción inteligente)
- MySQL (base de datos)
- FastAPI (API REST)
- HTML + Bootstrap (interfaz)

## 📊 Mercado Objetivo
- 117.000 empresas en Cali
- Sector comercio y gastronomía
- PyMEs con facturación DIAN

## 📁 Estructura
- /Artefactos → Interfaces web
- /Notas → Apuntes del curso
- /Recursos → Investigación de mercado

## Contexto del proyecto
- PyMEs colombianas en Cali
- Python 3.10, Windows 8.1
- MySQL para base de datos
